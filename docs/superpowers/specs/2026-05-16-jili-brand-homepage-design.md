# 几里造物品牌展示首页设计说明

## 项目目标

为 `几里造物` 制作一个可直接上线的单页品牌展示网站，用于建立品牌记忆、快速传达业务范围，并引导潜在客户发起联系。

该页面不承担案例展示或复杂内容承载职责，而是作为品牌入口页使用。

## 品牌定位

几里造物是一家聚焦文旅空间升级改造、空间设计、文创产品开发设计、文化提炼与系统设计的文化科技品牌。

首页需要传达的核心认知：

- 几里造物不是普通装修或单点视觉设计服务
- 几里造物强调文化、空间、产品与系统的一体化整合
- 几里造物的价值在于让文化成为可传播、可记忆、可带走的空间资产

## 目标用户

主要受众为有空间升级、文旅更新、文化场景打造、品牌文化表达需求的潜在合作方，包括：

- 文旅项目方
- 商业空间经营者
- 民宿、酒店、文化消费场景运营者
- 需要文化提炼与系统设计支持的品牌客户

## 页面目标

页面需要在短时间内完成三件事：

1. 让访问者记住 `几里造物`
2. 让访问者快速理解业务方向
3. 让访问者能够直接联系

## 设计原则

- 极简：内容克制，不做传统企业官网式堆叠
- 记忆性：品牌名和一句品牌主张形成首页核心视觉
- 文化感：页面气质应具有艺术、科技、文化的交叉感
- 专业性：虽然极简，但要让访客明确感知到系统化能力
- 转化导向：页面底部提供清晰联系路径

## 信息架构

页面采用单页结构，共四个部分。

### 1. 首屏 Hero

内容：

- 品牌名：`几里造物`
- 品牌主张：`让文化成为空间资产`
- 补充说明：聚焦文旅空间升级改造、空间设计、文创产品开发、文化提炼与系统设计
- 主按钮：`发起合作`
- 次按钮：`了解业务`

目标：

- 用最短时间建立品牌记忆
- 明确传达业务边界
- 保持强留白和高辨识排版

### 2. 能力区

展示四个能力模块，每项配一句简短说明：

- 文旅空间升级改造
- 空间设计与场景叙事
- 文创产品开发设计
- 文化提炼与系统设计

目标：

- 让访客快速理解服务范围
- 避免长段文字造成阅读阻力

### 3. 差异化区

提炼自公司简介的三项核心优势：

- `文化即传播`
- `轻量改造`
- `全链条整合`

说明方向：

- 强调文化表达不止于装饰，而是具备传播性和记忆点
- 强调在不大拆大改的前提下实现空间价值提升
- 强调从文化提炼、空间表达、文创延展到系统落地的一体化能力

### 4. 联系区

内容：

- 一句合作引导文案
- 微信
- 电话
- 邮箱

目标：

- 让访问者在情绪和认知建立后顺畅完成联系动作

## 视觉方向

整体风格定义为：`极简 + 艺术 + 科技 + 文化`

具体执行原则：

- 以大字号中文品牌名作为主视觉锚点
- 使用低饱和中性色体系，避免廉价商务风
- 通过留白、细线、秩序化网格和节制的动效表达科技感
- 通过文字节奏与版式比例表达文化感和展陈感
- 不使用通用企业官网卡片堆砌式视觉

## 动态方向

页面加入 `克制、轻量、细腻` 的动态 UI，用于增强品牌气质，不以炫技为目的。

动态范围包括：

- 首屏品牌名、主张、副文案的分层入场
- 背景中极轻的网格、噪点、渐变光感或扫描式流动
- 能力区与差异化区在滚动进入视口时的渐显与轻微位移
- 按钮、链接、联系方式的精细悬停反馈

动态范围不包括：

- 重型 3D 场景
- 高密度粒子特效
- 强烈视差
- 自动播放视频背景
- 影响阅读和加载速度的复杂动画

目标：

- 保持首页的高级感与呼吸感
- 让科技感通过秩序和节奏体现，而不是通过夸张视觉体现
- 确保移动端也能稳定呈现

## 文案方向

文案语气不采用传统销售口吻，而采用品牌声明式表达。

首页核心文案基于以下认知展开：

`我们用在地文化为商业空间注入灵魂，用设计让文化成为可传播、可记忆、可带走的空间资产。`

网页文案将压缩为更适合品牌首页的表达，例如：

`以在地文化为根，以设计为形，重塑空间、产品与体验之间的关系。`

## 实现范围

本轮实现包含：

- 一个可独立部署的品牌展示首页
- 适配桌面与移动端
- 适合后续部署到 Cloudflare Pages

本轮不包含：

- 案例展示系统
- 后台管理
- 表单提交后端
- 多页面官网结构
- CMS

## 成功标准

- 首页在视觉上能形成鲜明品牌记忆
- 用户可以在首屏和后续极少滚动中理解品牌与业务
- 页面结构足够轻，适合作为品牌入口页长期使用
- 页面代码可直接用于静态托管上线

## 风险与边界

- 若缺少最终联系信息，联系区需先用占位内容
- 若后续新增案例、团队、文章等内容，单页结构可能需要扩展为多页面网站
- 视觉表达需避免过度艺术化导致业务信息模糊
